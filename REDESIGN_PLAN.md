# NILO Website Redesign - Stumptown Style

## Goal
Redesign NILO website (watch e-commerce) to match Stumptown Coffee's design aesthetic and user experience.

## Design Elements from Stumptown

### Color Palette
- **Primary Background**: Cream/Beige (#F5F3EF)
- **Accent Color**: Deep Green (#4A5D4F)
- **Secondary Accent**: Warm Red/Orange
- **Text**: Dark Brown/Black
- **White**: Clean whites for contrast

### Layout Features
1. **Top Banner**: Promotional announcements (rotating)
2. **Navigation**: 
   - Clean horizontal nav
   - Logo center
   - Search, Account, Cart icons on right
   - Sticky navbar
3. **Hero Section**:
   - Large lifestyle images
   - Overlay text
   - CTA buttons
4. **Product Grid**:
   - Clean product cards
   - Hover effects
   - Product images with descriptions
5. **Featured Sections**:
   - Split layouts (image + text)
   - Holiday/seasonal promotions
6. **Footer**:
   - Multi-column layout
   - Newsletter signup
   - Social media links

### Typography
- Modern sans-serif fonts
- Bold headings
- Clean hierarchy
